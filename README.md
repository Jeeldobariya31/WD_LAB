## üìò Overview
This repository contains all 26 practical experiments from the **Web Designing Laboratory (2301CS202)** course conducted in the **2nd Semester (A.Y. 2024‚Äì25)** at **Darshan University**.  
Each practical helps students understand and implement **HTML, CSS, and Bootstrap** concepts for developing visually appealing and responsive web pages.

---

## üß© Practical List (As per Lab Manual)

| No. | Title | Description |
|-----|--------|-------------|
| **1** | Basic Structure of HTML | Create a simple web page using HTML structure tags (`<html>`, `<head>`, `<body>`). |
| **2** | Formatting Tags | Practice text formatting using tags like `<b>`, `<i>`, `<u>`, `<mark>`, `<small>`, etc. |
| **3** | Paragraphs and Headings | Demonstrate various heading levels and paragraph formatting. |
| **4** | Ordered, Unordered & Definition Lists | Create and style different types of lists for displaying information. |
| **5** | Hyperlinks | Implement navigation using `<a>` tags and internal/external linking. |
| **6** | Image and Marquee | Embed images, scrolling text, and background properties in a webpage. |
| **7** | Tables in HTML | Create tables with `rowspan`, `colspan`, and border customization. |
| **8** | Frames & Iframes | Divide web pages into multiple sections using `<frameset>` and `<iframe>`. |
| **9** | Forms in HTML | Build a student registration form using `<input>`, `<select>`, `<textarea>`, and form attributes. |
| **10** | Multimedia Integration | Insert video and audio files using `<video>` and `<audio>` tags. |
| **11** | Internal and Inline CSS | Demonstrate inline, internal, and external CSS styling on web pages. |
| **12** | CSS Selectors and Properties | Explore CSS selectors, colors, borders, and background customization. |
| **13** | CSS Layout | Implement layout control using `float`, `position`, and display properties. |
| **14** | CSS Box Model | Practice margin, padding, border, and content area customization. |
| **15** | CSS Animation | Add animations using `@keyframes`, transitions, and transforms. |
| **16** | Pseudo Classes and Elements | Explore interactive CSS effects using pseudo-selectors like `:hover`, `:focus`, and `:before`. |
| **17** | Responsive Design Introduction | Learn about viewport settings and basic responsive layout design. |
| **18** | Bootstrap Introduction | Use Bootstrap CDN and grid system for responsive structure. |
| **19** | Bootstrap Typography & Utilities | Use Bootstrap classes for text formatting and layout spacing. |
| **20** | Bootstrap Components | Implement Navbar, Buttons, Cards, and Forms using Bootstrap. |
| **21** | Bootstrap Grid Layout | Build a multi-column responsive layout using the Bootstrap grid system. |
| **22** | Forms with Bootstrap | Design a student registration or contact form using Bootstrap form controls. |
| **23** | Bootstrap Modal and Alerts | Use Bootstrap components like modals, alerts, and progress bars. |
| **24** | Mini Project ‚Äì Portfolio Page | Create a personal portfolio page with sections like About, Skills, and Contact. |
| **25** | Mini Project ‚Äì Product Page | Build a product listing page with responsive cards and image grids. |
| **26** | Final Project ‚Äì Responsive Website | Develop a multi-page responsive website combining all learned concepts (HTML, CSS, Bootstrap). |
